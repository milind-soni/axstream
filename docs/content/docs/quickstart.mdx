The repo ships a reference implementation in Python: the newline-committed
stream compiler, action catalog, accessibility-tree observer, and pipelined
executor. Manage the environment with
[uv](https://docs.astral.sh/uv/).

## 1. Dry demo — no keys, no server

```sh
uv run --with pytest python demo_dry.py
```

Replays a canned LLM response at realistic decode speed into the real compiler
and executor against a mock computer. It prints a timeline of actions executing
mid-stream, then compares streamed total vs a buffered baseline (wait for the
full response, then act) — typically ~37% saved.

## 2. Live streaming (macOS)

The live path drives [cua](https://github.com/trycua/cua)'s `computer-server`
for clicks, keys, and the accessibility tree.

<Steps>

<Step>

### Start computer-server

Grant your terminal **Accessibility** permission first.

```sh
git clone https://github.com/trycua/cua ../cua
cd ../cua/libs/python/computer-server
uv run python -m computer_server --port 8765
```

</Step>

<Step>

### Stream a task

```sh
export GROQ_API_KEY=...   # or ANTHROPIC_API_KEY with --provider anthropic
uv run python demo_live.py --task "open TextEdit and type hello world" \
  --uri ws://localhost:8765/ws
```

Flags: `--provider` (groq / anthropic / openai), `--model`, `--max-bursts`,
`--no-risky` to block actions marked `risk:risky`.

</Step>

</Steps>

## 3. Learn-and-replay (the instant tier)

`demo_learn.py` shows the three-speed system: the first time you type a
command, an LLM plans and executes it (~0.5–1s) and the run is captured as a
parameterized macro; from then on, a local 350M matcher replays it in ~100ms
with no LLM — including slot variations ("titled standup" → "titled yoyo").

Extra prereq — a tiny matcher on `:8791` (or set `AXSTREAM_TINY_URL`):

```sh
llama-server -m <matcher>.gguf --port 8791 -ngl 99 -c 4096 --no-webui
uv run python demo_learn.py
```

<Callout title="Use a fine-tuned matcher">
The base LFM2.5-350M scores ~47% end-to-end on match + slot extraction; a
small LoRA fine-tune (~2.5k synthetic examples, under an hour on Apple
Silicon) reaches ~93% at the same ~100ms. Misses are safe — a failed match
falls back to the LLM tier — but the instant tier only feels instant with a
tuned matcher.
</Callout>

## 4. Tests

```sh
uv run pytest
```
